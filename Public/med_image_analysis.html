Medical Image Analysis


Worked on four Machine Learning models to study the best accuracies initially on four datasets of Kidney Tumor, Pneumonia, Covid-19 detection and Brain Tumor Detection. Brain Tumor Detection was finally chosen to be worked on in Phase-II on four different models using CNN.

  • Python libraries such as tensorflow, keras, pandas
  • CNN architectures - ResNet50, VGG16, MobileNet
  • SVM Classifier and fine-tuning
